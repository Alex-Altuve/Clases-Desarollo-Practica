# Clases de Desarollo - Practica 

## Intallation

```
$ npm install
```
## Running de app

1. Run the command for transpilation to JS
```
$ node run build 
```

2. Move to one direcotory and execute the following command

```
$ cd src\patrones\Creacionales
$ ts-node 01-Factory-Method.ts

```
